# Career Masters Website

A modern website for Career Masters, an international student services company.

## Features

- Responsive design for all devices
- Dark mode support
- Interactive country selection
- Service showcase
- Testimonials and success stories
- Contact form
- FAQ section

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Lucide React icons
